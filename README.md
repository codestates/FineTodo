
📰 FineAcademy
우리 아이를 위한 학원찾기 서비스!


🏠 FineAcademy Homepage


✨ Feature

우리집 주변의 학원, 교육기관등의 정보를 알려드립니다.

필요한 과목과 학년 수준등을 선택하여 찾을수 있습니다.

차트를 통해 교육 키워드와 학생들의 관리현황을 시각적으로 확인할수 있습니다. 

매주 우리 아이의 현재 학습진단과 수준을 진단하여 제공해드립니다.
